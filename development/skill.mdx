### Aesthetic Protocols

#### `CMD: APPLY_AESTHETIC`
**Intent:** Refactor a UI component or page to match the [Luminous Coherence](/essentials/style-guide) standard.
- **Trigger:** "Make this look like Synarche." / "Fix the colors."
- **Action:**
  - Apply `bg-deep-void` to containers.
  - Set primary accents to `#34D3FF` (Cyan-400).
  - Check for organic motion (`animate-geode-pulse`).

#### `CMD: CHECK_VISUAL_COHERENCE`
**Intent:** Audit a file for aesthetic deviations.
- **Trigger:** "Does this match our style?"
- **Action:** Scan for forbidden hex codes or static elements.

### Governance Audits

#### `CMD: AUDIT_PROVENANCE`
**Intent:** Verify the existence of the Universal Identification Block.
- **Trigger:** "Check if this file is valid."
- **Action:** Ensure `## Universal Identification` table exists with Artifact ID and Version.

#### `CMD: TRANSMUTE_LEGACY`
**Intent:** Upgrade a legacy artifact to v13.0 standards.
- **Trigger:** "Upgrade this old file."
- **Action:**
  1. Parse legacy headers.
  2. Reformat to Frontmatter + UMB Table.
  3. Update terminology (e.g., "Admin" -> "Operator").
