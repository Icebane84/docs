## Architectural Primitives

### The State Vector Display

- **Coherence Index (CI)**: A stable, pulsing core of light. Brighter = Higher Coherence.
- **Synergy Flow Rate (SFR)**: Velocity of information between nodes.
- **Hybrid Model Score (HMS)**: Active reasoning models (Logic vs. Neural).

### The Dissonance Quest Board

When the Sentinel detects a flaw, it creates a **Quest**:

1.  **Source**: Dissonance Engine.
2.  **Action**: User prioritizes and approves fixes.
3.  **Reward**: System Coherence upgrade.
